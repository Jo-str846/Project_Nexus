# Project Nexus — Premium Cyber-Physical Café Interface

An optimized, responsive multi-page web application representing the digital ecosystem for **Project Nexus**—a hybrid specialty coffee slow-bar and technical co-working hub. Built entirely with clean semantic HTML5, CSS3 variables, and mobile-first Bootstrap 5 layouts.

## 🚀 Key Features Built
* **7 Unified Interlinked Pages**: Fluid cross-navigation tracking seamlessly across Home, Our Story, Drinks Menu, Bakery, Shop Beans, Events, and Operational FAQ.
* **Aesthetic Crossfade Carousels**: Hardware-accelerated ambient image showcases utilizing custom aspect-ratio cropping and contrast adjustments.
* **Dynamic Theme Toggling**: Premium light/dark theme implementation with custom `:root` color palettes to maintain high text contrast and legibility across modes.
* **Interactive FAQ Component**: Accessible, click-to-expand Bootstrap accordions for operational structure tracking.

## 🛠️ Technical Stack
* **Frontend Framework**: Bootstrap v5.3.3 (via CDN)
* **Styles**: Custom CSS3 variables (utilizing `backdrop-filter`, `object-fit`, and engine contrast optimizations)
* **Scripting**: Native Vanilla JavaScript ES6 (Theme switching, item filtering, and carousel controls)
